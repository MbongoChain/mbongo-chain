# Mbongo Chain — Roadmap  
Status: Canonical  
Version: v1.0

This roadmap outlines planned milestones for the evolution of Mbongo Chain.  
It is divided into short-term, mid-term, and long-term development phases.

---

# 1. Short-Term (0–12 months)

## 1.1 Core Protocol  
- Finalize PoS + PoUW consensus (PoX)  
- Release main Rust node  
- Implement compute marketplace MVP  
- PoUW CPU execution (fallback path)  
- Transaction pipeline: mempool → execution → commit  
- AIDA v1 (bounded economic controller)

## 1.2 Developer Ecosystem  
- TypeScript SDK v1  
- Rust SDK v1  
- RPC API v1  
- CLI wallet v1  
- Documentation hub  
- Local devnet launcher  

## 1.3 Governance  
- DAO v1  
- Founder Council activation (10-year timer)  

---

# 2. Mid-Term (12–36 months)

## 2.1 Compute Upgrades  
- GPU compute orchestration  
- Multi-GPU support  
- Redundant execution network  
- Fraud proofs v2  
- PoC hardware attestation v1  
- ZK-assisted compute verification

## 2.2 Runtime Upgrades  
- WASM smart contract engine v1  
- Deterministic JIT compiler  
- C-Gas pricing stabilization  

## 2.3 Network & Scaling  
- libp2p-based evolution: optimized streams  
- PoUW compute subnets  
- Compute sharding  
- Light client protocol  

## 2.4 AIDA Evolution  
- AIDA v2 (risk models, forecasting)  
- AIDA simulation sandbox  
- AIDA open data explorer  

---

# 3. Long-Term (36 months and beyond)

## 3.1 Advanced Compute Economy  
- On-chain AI model hosting  
- Model marketplace  
- GPU/TPU/NPU abstraction layer  
- Autonomous compute clusters  
- Cross-chain compute bridges  

## 3.2 Zero-Knowledge & Privacy  
- ZK proofs for compute jobs  
- Privacy-preserving inference  
- Confidential compute tasks (TEE + ZK hybrid)

## 3.3 Multiverse Architecture  
- Multi-runtime environment  
- App-specific compute chains  
- AI-native supernet with model replication  
- Global verifiable compute marketplace  

---

# 4. Vision Horizon (10–25 years)

- Mbongo as a global decentralized compute fabric  
- Trustless AI infrastructure powering real applications  
- Industry-standard verifiable compute protocol  
- Neutral, open, time-backed digital infrastructure  
- Evolution of PoUW into energy-efficient global compute grid
